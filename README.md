# book-store-backend
API for Book Store 

A brief overview of book store possibilities :
- Sign in / Sign up via Google , Facebook
- Buy books 

Entities :
- Book
- Genre
- Author 
- Orders
- Order detail
- Comment
- User

Roles : 
- User :
- Administrator : 
  
User can :
- Register 
- Buy book
- Create comment 
- Evualate a book

Administrator can :
- Sign in
- CRUD for entities
- Delete user
