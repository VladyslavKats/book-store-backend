# book-store-backend
API for Book Store 

A brief overview of book store possibilities :
- Sign in / Sign up via Google , Facebook
- Buy books 

Entities :
- Book
- Genre
- Author 
- Orders
- Order detail
- Comment
- User

Roles : 

- User :
  a) Register 
  b) Buy book
  c) Comment 
  d) Evualate a book
  
- Administrator : 
  a) Sign in
  b) CRUD for entities
  c) Delete user
